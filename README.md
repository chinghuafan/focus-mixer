# 專注高效：情境混合器 (Focus Mixer)

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
## 📝 專案介紹

這是一個簡約、開源的網頁應用程式，旨在提供一個沉浸式的氛圍空間，結合環境背景、背景音樂串流，以及番茄鐘計時器，幫助使用者提升專注力，進入心流狀態。

## ✨ 主要功能

* **沉浸式背景切換：** 快速切換多個靜態或動態背景情境。
* **YouTube 氛圍音樂：** 透過 YouTube Iframe API 串流背景音樂播放清單。
* **番茄鐘計時器：** 提供可自訂的專注時長（25/45/15 分鐘等），包含 **啟動、暫停、重設** 功能。
* **極簡介面：** 專注模式下，只顯示核心計時器，隱藏即時時鐘，減少干擾。
* **音量控制：** 獨立控制背景音樂音量。

## 🛠️ 技術棧

* **HTML5 / CSS3：** 基礎結構與風格。
* **Vanilla JavaScript：** 專案的核心邏輯，包括計時器和狀態管理。
* **YouTube Iframe API：** 處理背景音樂的載入與播放。

## ⚙️ 如何運行 (Run Locally)

這個專案是純前端應用，不需要後端伺服器。

1.  **複製程式碼：**
    ```bash
    git clone [https://github.com/chinghuafan/focus-mixer.git](https://github.com/chinghuafan/focus-mixer.git)
    cd focus-mixer
    ```
2.  **打開檔案：** 直接在您的瀏覽器中雙擊打開 `index.html` 即可運行。
